Projetos

Uma página de homenagem

uma página da Web escrevendo sobre essa pessoa adicionando sua imagem. No topo da página, adicione a imagem e o nome da pessoa e abaixo disso forneça o layout para o restante dos detalhes. Você pode usar parágrafos, listas, links, imagens com CSS para dar uma aparência descendente.

Site Parallax

Um site de paralaxe inclui imagens fixas em segundo plano que você pode manter no lugar e pode rolar a página para baixo para ver diferentes partes da imagem. Com conhecimento básico de HTML e CSS, você pode dar um efeito de paralaxe a um site.

Página inicial

Uma página de destino é outro bom projeto que você pode fazer usando HTML e CSS, mas requer um conhecimento sólido desses dois blocos de construção. Você usará muita criatividade ao criar uma página de destino. Você vai praticar como adicionar rodapé e cabeçalho, criar colunas, alinhar itens, dividir as seções e muitas outras coisas.

Site do restaurante

uma bela página da web para um restaurante. Fazer um layout para um restaurante será um pouco complicado do que os exemplos de projetos anteriores. Você irá alinhar os diferentes alimentos e bebidas usando uma grade de layout CSS. Você adicionará preços, imagens e também precisará dar uma aparência bonita, usando a combinação adequada de cores, estilo de fonte e imagens.
